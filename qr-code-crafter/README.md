# QR Code Crafter

A professional QR code generator that creates QR codes instantly for any text or URL.

**Live Demo:** [vampishwolf.github.io/qr-code-crafter](https://vampishwolf.github.io/qr-code-crafter/)

## Features

- Generate QR codes from any text or URL
- Multiple download formats (PNG, JPEG)
- Customizable sizes
- Clean, responsive interface

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS

## Usage

Visit the live site, enter your text or URL, and download your QR code.
