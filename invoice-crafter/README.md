# Invoice Crafter

A professional invoice generator with PDF export, multiple currency support, and live preview.

**Live Demo:** [vampishwolf.github.io/invoice-crafter](https://vampishwolf.github.io/invoice-crafter/)

## Features

- Create professional invoices
- PDF export functionality
- Multiple currency support
- Live preview as you type
- Customizable templates
- Add multiple line items
- Automatic calculations

## Technologies

- JavaScript
- jsPDF (for PDF generation)
- HTML
- CSS

## Usage

1. Fill in your business details
2. Add client information
3. Add line items with descriptions and prices
4. Preview your invoice
5. Download as PDF
