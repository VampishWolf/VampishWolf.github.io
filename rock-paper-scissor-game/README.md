# Rock Paper Scissors

A classic rock-paper-scissors game where you play against the computer.

**Live Demo:** [vampishwolf.github.io/rock-paper-scissor-game](https://vampishwolf.github.io/rock-paper-scissor-game/)

## Features

- Play 5 rounds against the computer
- Score tracking with visual scoreboard
- Win/loss/tie detection
- Play again option

## Technologies

- JavaScript
- HTML
- CSS

## How to Play

1. Open the page
2. Open browser console (Ctrl+Shift+K or Cmd+Option+J)
3. Type `play()` to start the game
4. Enter your choice (rock, paper, or scissors) when prompted
5. Play through 5 rounds and see who wins!
